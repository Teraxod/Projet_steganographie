🕵️‍♂️ Portail Stéganographie

Bienvenue sur l'interface Apache dédiée à la découverte et la manipulation de la stéganographie. Cette page propose plusieurs démonstrations pour cacher et révéler des informations secrètes de différentes manières.

🖼️ Section Image :

  • Cacher un mot dans les métadonnées EXIF d'une image
  • Extraire un flag des métadonnées EXIF d'une image
  • Cacher un flag dans les strings d'une image
  • Trouver un flag dans les strings d'une image
  • Cacher une image dans une image
  • Extraire une image cachée dans une image
  • Cacher un flag encodé texte
  • Décoder un flag caché
  • Cacher un fichier texte dans une image
  • Extraire un fichier texte caché dans les métadonnées d’une image

🔊 Section Son :

  • Cacher un mot dans du son
  • Extraire un mot d'un fichier son

🤖 Section IA :

  • Faire un acrostiche
  • Créer une image
  • Cacher un flag en lsb
  • Décoder lsb
  • Cacher un flag en whitespace
  • Décoder un flag en whitespace
